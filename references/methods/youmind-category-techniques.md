# GPT Image 2 - Category-Specific Techniques

> 不同应用场景的 prompt 技巧总结

---

## Profile / Avatar

**场景定位**: 人物肖像和头像生成

### 关键技巧

- 使用具体的人物描述（年龄、发型、服装、表情）
- 指定摄影参数（景深、布光、焦段）
- 定义场景和氛围（室内/室外、季节、天气）
- 使用 mirror selfie、phone screen 等创意构图
- 参考特定摄影风格（editorial, snapshot, cinematic）

## Social Media Post

**场景定位**: 社交媒体内容创作

### 关键技巧

- 强调视觉冲击力和第一眼吸引力
- 营造 lifestyle 氛围和场景感
- 模仿特定社交平台的内容风格
- 常有 mood board / collage 风格需求
- 注重色彩搭配和整体调性

## Infographic / Edu Visual

**场景定位**: 信息图表和教育可视化

### 关键技巧

- 使用结构化 JSON 定义信息层级
- 明确标注线、说明文字的位置和内容
- 灵活运用对比、时间线等图表形式
- 日式"Ponchi-e"风格的信息图设计
- 菜谱卡、产品对比、制作流程等模板化内容

## YouTube Thumbnail

**场景定位**: YouTube 缩略图生成

### 关键技巧

- 高对比度、大字体、强烈视觉冲击
- 人物表情夸张、动作戏剧化
- 文字和图像的比例关系要平衡
- 模仿特定频道或风格的缩略图设计
- 通常需要特定的宽高比（16:9）

## Comic / Storyboard

**场景定位**: 漫画和故事板创作

### 关键技巧

- 使用 panel 结构定义分镜
- 指定画风（日式漫画、美式漫画、素描等）
- 对话框和文字气泡的精确布局
- 角色在多个分镜中的一致性要求
- 动作线和特效的添加

## Product Marketing

**场景定位**: 产品营销和广告设计

### 关键技巧

- 高端摄影棚布光参数
- 品牌色和品牌元素的精确控制
- 产品和模特的互动构图
- 杂志编辑风格的排版
- 生活方式场景的融入

## E-commerce Main Image

**场景定位**: 电商主图/详情图

### 关键技巧

- 纯白/干净背景的产品展示
- 多角度产品展示和细节特写
- 爆炸图/结构图展示产品内部
- scene-in-scene 的场景化展示
- 精确的品牌和规格信息文字渲染
