## 🏗️ 核心方法论速览

### 1. JSON 结构化 Prompt
将复杂布局分解为层级化的 JSON schema，精确控制每个元素的属性。最适合：海报、信息图、UI 设计。

### 2. 自然语言描述式
像 brief 一样用自然语言描述，加入摄影/艺术术语提升质量。最适合：摄影、插画、概念艺术。

### 3. 动态参数化
使用 `{argument}` 语法将 prompt 模板化，支持一键变体生成。配合 Raycast 效率最高。

### 4. 摄影级参数
精确控制布光、景深、构图、相机参数，获取商业级图像质量。

### 5. 负向提示约束
通过 `Avoid...` 排除不需要的元素，提升输出质量和一致性。
